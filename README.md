# Maratona Live Code Totvs - Chat API

> Projeto do back-end criado durante o evento Maratona Live Code Totvs

## Tecnologias

- Typescript
- Nest Js
- Rest API
- Web Sockets
- OpenApi
- Docker

## Docker

### run API

- docker-compose -f docker/docker-compose.yml up -d

### stop API

- docker-compose -f docker/docker-compose.yml down
